# 世界觀設定 (World Building)

## 1. 核心概念 (Core Concept)
*   **一句話梗概 (Logline)**：一名凡人駭客穿越到一個被深淵侵蝕、歷史被篡改的修仙世界，在眾人皆以「向下飛升」墮入深淵為榮的瘋狂中，他必須藉助能轉化深淵之力的賽博系統，駭入這個世界的源代碼，扭轉錯誤的真理。
*   **核心衝突**：
    *   **表層**：主角唯一清醒 vs 全世界集體瘋狂。
    *   **深層**：冰冷的機械邏輯 (System/Order) vs 扭曲的血肉深淵 (Abyss/Chaos)。

---

## 2. 扭曲的真理 (The Twisted Truth)

### 2.1 背景概述
*   深淵入侵後，修仙界歷史被改寫，**「變異」=「進化」，「瘋狂」=「頓悟」**。
*   天空是紫紅色的，月亮散發詭異的紫色光芒。
*   植被大多是黑色的，經常發出詭異的聲響。

### 2.2 關鍵設定 - "飛升" (Ascension)
*   **世人認知**：飛升 = 向下深入地底。越接近地底深淵，靈氣越濃（其實是瘴氣）。
*   **真相**：這是獻祭給古老邪神「深淵母神」的過程。

### 2.3 修煉真相
*   **本質**：修煉 = **主動異變** + **精神崩壞**。
*   **限制器 (The Limiter)**：植入體內的「鎮魂釘」，讓人產生幻覺，以為自己修的是仙氣，其實身上長滿了毛。
*   **等級對照**：
    1.  **寄生期 (Parasitic Stage)** / 練氣：深淵孢子寄生。
    2.  **異結期 (Node Formulation)** / 築基：體內長出異體器官。
    3.  **裂變期 (Fission)** / 金丹：肉體不穩定分裂。
    4.  **不可名狀期 (The Unspeakable)** / 元嬰：脫離人形。

### 2.4 特殊術法 (New)
*   **追魂引 (Soul Tracker)**：內門高階追蹤術。施術者可在目標身上留下隱形標記，300公里內精確定位。主角目前已被標記。
*   **血緣感應 (Bloodline Sensing)**：特殊血脈能力，可感知親緣關係或接觸過親緣之人的氣息（如錢執事）。

## 3. 勢力戰國 (Factions)

除了長生宗之外，世界上還存在其他被深淵影響或對抗深淵的勢力：

*   **東方深淵 (肉體恐怖)**：
    *   **血肉衍宗**：追求肢體增生、無限再生。
    *   **千手慈航**：信仰「多即是美」，崇拜多肢體形態。
*   **西方深淵 (精神恐怖)**：
    *   **星智教派**：追求宇宙真理，但所謂真理來自瘋狂的星空。
    *   **深黯教會**：崇拜深海恐懼，追求夢境低語與觸手。
*   **外來虛空 (存在抹除)**：
    *   **虛無宗**：追求萬物歸零，認為只有徹底的虛無才能對抗深淵，與深淵勢力敵對。

## 4. 長生宗 (Changsheng Sect)

### 4.1 宗門概述
*   長生宗的真正目的並非修煉飛升，而是**育種**。
*   整個宗門就是一個巨大的培養皿，所有弟子都是用來培育「深淵種子」的養料。
*   那些所謂的「飛升」，其實是被獻祭給深淵母神的儀式。
*   高層修士們之所以能夠長生，是因為他們已經和深淵母神簽訂了契約，成為了「祂」在這個世界的代言人。

### 4.2 宗門結構
```text
掌門 (元嬰期，從未露面)
    ├── 內門
    │   ├── 五名長老 (築基期以上)
    │   │   ├── 大長老
    │   │   ├── 二長老
    │   │   ├── 三長老 (周長老) ← 與外門有秘密交易
    │   │   ├── 四長老
    │   │   └── 五長老
    │   ├── 執法堂 (Law Enforcement Hall) ← 負責清理叛徒與「垃圾」，這次派出了殺手。
    │   └── 內門弟子
    └── 外門
        ├── 四名執事 (築基期)
        │   ├── 趙執事 (已死) - 築基中期，管理東區
        │   ├── 錢執事 - 築基後期，擅長追蹤，與趙執事有血緣關係
        │   ├── 孫執事 - 膀大腰圓的胖子，精明
        │   └── 李執事 - 最年輕但最強
        └── 外門弟子
            ├── 普通弟子
            └── 雜役弟子 (最底層)
```

### 4.3 外門地點
*   **外門弟子居住區**：雜役弟子居住的破舊區域。
*   **深淵區**：外門最危險的區域，專門清理深淵污染物，死亡率極高。
*   **後山**：變異獸出沒區域，是外門弟子心目中的禁地，也是主角的狩獵場。
*   **執事殿**：外門四執事辦公場所。
