第八章：蟄伏

走出深淵區的那一刻，刺眼的陽光讓李默下意識地瞇起了眼睛。
他已經在那片被黑霧籠罩的詭異地帶待了整整六個多小時，眼睛早就習慣了那種昏暗的環境。突然暴露在正常光線下，反而讓他有些不適應。

「嘶……」
李默用手遮住眼睛，讓瞳孔慢慢適應光線的變化。
與此同時，他的大腦也在飛速運轉。

現在的情況很微妙。
他已經殺掉了趙執事的「血肉分身」，按照記憶碎片中透露的訊息，那座肉山是趙執事為了突破瓶頸而煉製的「外丹」，用來囤積能量和維持修為。
現在外丹沒了，趙執事一定受到了不小的反噬。
輕則修為倒退，重則直接走火入魔。

但問題是，這老東西還沒死。
只要他還活著，就一定會來找自己的麻煩。

「系統，掃描周圍環境。」

【正在掃描...】
【掃描完成。】
【半徑500米範圍內未發現高威脅目標。】
【檢測到12個低威脅生命體 (外門弟子)。】
【檢測到1個中威脅生命體 (變異獸，正在進食)。】

只有外門弟子和變異獸？
趙執事沒來？
這有些出乎李默的預料。
按理說，自己的分身被幹掉了，正常人的反應應該是第一時間趕過來查看情況才對。
除非……

李默的眼睛微微眯起。
除非趙執事現在正在承受反噬，根本無暇分身。
又或者，他在憋大招。

無論是哪種情況，對李默來說都是好消息。
他需要時間。
時間來消化這次戰鬥獲得的收益，時間來提升自己的實力，時間來準備接下來的對決。

李默深吸一口氣，調整好自己的狀態，然後邁步走向肉田的外圍區域。
那裡依然有許多外門弟子在勤勤懇懇地幹活，給那些恐怖的「屍嬰草」施肥澆水。
他們渾然不知，就在幾百米外的深淵區，剛剛發生了一場足以改變整個外門格局的戰鬥。

「嘿，那不是李默嗎？」
一個正在搬運肥料桶的弟子注意到了李默的身影，驚訝地叫了起來。
「我操，他居然還活著？」

「不會吧？趙執事不是把他扔進深淵區了嗎？進去的人有幾個能活著出來的？」
另一個弟子也湊了過來，眼中滿是難以置信。

「會不會是他運氣好，躲在某個角落沒被那些怪物發現？」
「說不定吧，這小子命還真大。」
「可是你看他那副鬼樣子，渾身是血，衣服都爛了……肯定遇到了什麼可怕的東西。」

李默聽到了這些議論聲，但他沒有理會。
他的思緒完全沉浸在接下來的計劃中。

首先，他需要低調。
殺死趙執事的分身這件事，絕對不能讓任何人知道。
在外人看來，他只是一個運氣好、僥倖從深淵區活著出來的小雜魚。
這種低調的形象，正是他目前最需要的保護色。

其次，他需要找一個安全的地方閉關。
他現在的能量儲備高達235點（扣除解鎖記憶的50點後還有185點），這些能量如果全部用來修煉，足夠讓他再突破一到兩個小境界。
但修煉需要時間和安靜的環境，而外門弟子的居住區別說安靜了，恐怕連覺都睡不安穩。

第三，也是最重要的一點——他需要準備一個殺死趙執事的手段。
駭入協議雖然有效，但那只是一個Beta版本，對付肉山那種沒有智慧的本能生物還行，對付趙執事這種老奸巨猾的傢伙，恐怕不會那麼順利。
他需要更強的武器。

「嘿！李默！」
一聲呼喚打斷了他的思緒。
李默抬起頭，看到一個熟悉的身影正朝他走來。
是王大力。
這個半張臉被肉瘤佔據、左臂變成蟹鉗的彪形大漢，此刻正用一種複雜的眼神看著李默。

「我還以為你死定了呢。」
王大力走到李默面前，圍著他轉了一圈，語氣裡帶著幾分驚訝和一絲若有若無的忌憚。
「深淵區啊……那可不是一般的鬼地方。你是怎麼活下來的？」

「運氣。」
李默淡淡地說道，臉上露出一個疲憊的笑容。
「找了個石縫躲了一晚上，那些怪物沒發現我。」

「就這樣？」
王大力顯然不太相信，「那你身上這些血是怎麼回事？」

「之前在外圍區施肥的時候不小心被那些屍嬰草的根鬚刮傷的，一直沒來得及處理。」
李默輕描淡寫地編了個謊言，「王師兄，我現在有些累了，想先回去休息。」

王大力盯著李默看了好幾秒，那隻長在肉瘤上的小眼睛閃爍著複雜的光芒。
最後，他點了點頭。
「行，你去吧。不過……」
他壓低聲音，湊近李默的耳邊，「趙執事剛才臉色很難看，像是出了什麼大事。如果他找你麻煩，你自己小心點。」

「多謝師兄提醒。」
李默拱了拱手，然後快步離開了肉田。

趙執事臉色很難看？
看來分身被殺的反噬，比李默預想的還要嚴重。
這是好消息。
越是虛弱的敵人，就越容易對付。

李默沿著山路往回走，一路上遇到了不少外門弟子。
他們看向李默的眼神都很複雜——有好奇、有驚訝、有嫉妒，甚至還有幾分恐懼。
能從深淵區活著出來的人，要麼是運氣逆天，要麼是實力深藏不露。
無論是哪種，都不是他們這些普通弟子能夠招惹的。

李默對這些目光視若無睹，只是一步一步地往前走。
他的目的地不是外門弟子的居住區，而是後山的一處廢棄洞穴。
那是他在原主的記憶中發現的一個隱秘地點，據說曾經是某位前輩閉關修煉的靜室，後來那位前輩走火入魔爆體而亡，這裡就被廢棄了。
正因為有這個「不祥之地」的名聲，平時很少有人靠近，正好適合他閉關修煉。

穿過一片陰森的黑色樹林，李默終於來到了那處洞穴的入口。
洞口被一些雜草和藤蔓遮蔽，如果不仔細看，很難發現這裡還有一個洞穴。

「系統，掃描洞穴內部。」

【正在掃描...】
【掃描完成。】
【洞穴深度：約30米。】
【內部空間：約50平方米。】
【威脅評估：無高威脅生物。】
【發現低威脅生物：變異蝙蝠 x12 (可清除)。】
【發現物品：殘破法陣 x1、枯骨 x1、儲物袋 (已損壞) x1。】

有殘破法陣？還有儲物袋？
李默的眼睛亮了起來。
看來當年那位爆體而亡的前輩，還留下了一些遺產。

他小心翼翼地撥開洞口的藤蔓，彎腰鑽了進去。
洞穴內部一片漆黑，只有他眼中流動的藍色數據流提供了一些微弱的光亮。
系統的「解析之眼」自動開啟，將周圍的環境一一呈現在他的視野中。

洞穴的牆壁上爬滿了黑色的苔蘚，散發著一股潮濕的霉味。
地面上零星散落著一些骨頭——有人類的，也有變異獸的。
在洞穴的最深處，有一個用石頭搭建的簡易平台，平台上躺著一具白骨，旁邊還放著一個破爛的布袋。

那就是儲物袋了。
李默走過去，蹲下身，小心翼翼地撿起那個布袋。
袋子已經腐爛了大半，裡面的東西散落了一地。

【正在解析物品...】
【物品1：下品靈石 x5 (已污染，可通過生物反應爐淨化)】
【物品2：殘破玉簡 x1 (內含功法殘篇：血箭術)】
【物品3：腐爛丹藥 x3 (毒性：高，能量殘留：微量)】
【物品4：不明金屬碎片 x1 (材質：未知，疑似舊時代遺物)】

血箭術？
李默的眼睛猛地瞪大。
這不正是大綱裡提到過的那個法術嗎——需要消耗自身血液來發動的攻擊性法術！
如果能用系統對它進行改造……

他連忙拿起那枚殘破的玉簡，將其貼在額頭上。
一股陌生的信息流湧入腦海，但在系統的過濾和翻譯下，這些信息被轉化成了他能夠理解的格式。

【正在解析功法...】
【解析完成。】

【功法名稱：血箭術 (殘篇)】
【等級：下品】
【消耗：自身精血 10ml / 次】
【威力：中等 (對練氣期有效)】
【副作用：長期使用會導致氣血虧損】
【備註：這是一種典型的「傷敵一千，自損八百」的法術。施法者需要割破手指，用自己的鮮血凝聚成箭，然後發射出去。】

【系統提示：檢測到功法可進行代碼修改。】
【是否開啟「編譯器」模組進行重寫？】
【注意：當前使用者境界為練氣二層，已達到編譯器初級權限的解鎖條件。】

可以修改了！
李默興奮地握緊拳頭。
之前在第四章的時候，他嘗試用系統升級骨劍，但因為等級不夠而失敗。
現在他已經突破到練氣二層，終於可以使用「編譯器」功能了！

「系統，開啟編譯器模組。」

【編譯器模組啟動中...】
【初始化完成。】
【歡迎使用「指令重寫」功能。】
【請選擇要修改的代碼片段。】

李默看著眼前浮現出的「血箭術」核心代碼，嘴角勾起一抹笑容。
這個法術的核心邏輯很簡單：消耗自身血液 → 凝聚成箭 → 發射。
問題出在第一步——消耗自身血液。
這讓施法者每次使用都要付出血的代價，用多了人就廢了。

但如果……把「自身血液」改成「敵人血液」呢？

李默開始在腦海中構思修改方案。
原本的代碼邏輯是這樣的：

【原始代碼：】
function 血箭術() {
    var 血液來源 = 施法者.自身;
    var 消耗量 = 10ml;
    抽取(血液來源, 消耗量);
    凝聚成箭();
    發射();
}

他需要做的，就是把「血液來源」從「施法者.自身」改成「目標.敵人」。
但這裡有一個問題——怎麼抽取敵人的血液？
敵人又不會乖乖站著讓你抽血。

李默思考了一會兒，想到了一個辦法。
他可以先用這個法術造成一個傷口，然後再從傷口中吸取血液來發動第二次攻擊。
這樣一來，第一箭可能還是要用自己的血，但之後的每一箭都可以用敵人的血來發動！

「系統，嘗試重寫代碼。」

【正在重寫...】
【警告：代碼修改可能導致法術不穩定。】
【是否繼續？】

「繼續。」

【執行中...】
【修改參數：血液來源 = 施法者.自身 || 目標.開放性傷口】
【新增功能：吸血箭 (當目標存在開放性傷口時，可直接抽取目標血液發動攻擊)】
【編譯完成。】
【新法術已生成：吸血箭術 (改)】

【吸血箭術 (改)】
【等級：下品 (存在升級潛力)】
【消耗：首次使用消耗自身精血 5ml，之後可從敵人傷口處吸取血液發動】
【威力：中等 -> 中高等 (連續攻擊時傷害遞增)】
【特殊效果：吸血 (每次命中從目標身上回收少量能量)】
【副作用：首次使用仍會造成少量氣血虧損】

「完美。」
李默滿意地看著這個新法術的屬性面板。
雖然首次使用還是要付出一點代價，但之後就可以用敵人的血來打敵人了。
不僅如此，這個法術還附帶了「吸血」效果，每次命中都能回收少量能量，簡直就是為他量身定做的！

有了這個法術，他對付趙執事的把握又大了幾分。

李默收起玉簡，開始清理洞穴。
那些變異蝙蝠被他用一個小型的數據脈衝轟成了灰燼，那具白骨也被他恭恭敬敬地埋在了洞穴外面。
雖然不知道這位前輩是誰，但人家的遺產對他幫助很大，最起碼要表示一下尊重。

清理完畢後，李默盤腿坐在石台上，開始準備閉關修煉。
他現在還有185點能量，足夠支撐一次長時間的閉關。
他的目標是在最短的時間內，把境界提升到練氣三層，甚至練氣四層。
只有這樣，他才有和趙執事正面對抗的資本。

「系統，開始修煉模式。」
「將所有能量轉化為境界提升。」

【收到指令。】
【正在分配能量...】
【預計可提升境界：練氣二層 -> 練氣三層 (需消耗100點能量)】
【剩餘能量：85點 (可用於後續提升或緊急情況)】
【修煉開始。】
【預計用時：12小時。】

李默閉上眼睛，感受著體內的能量開始流動。
純淨的力量從丹田湧出，沿著經脈流遍全身，不斷衝擊著修為的瓶頸。
這種感覺很奇妙——既有突破帶來的酸脹感，又有力量增長帶來的充實感。

洞穴外，夜色漸漸降臨。
山風從洞口吹進來，帶著一股寒意。
但李默渾然不覺。
他的意識已經完全沉入了修煉之中，與系統融為一體，專注於境界的突破。

這一夜，注定不平靜。
而一場更大的風暴，正在悄然醞釀。

在長生宗的執事殿中，趙執事正盤腿坐在一張鋪滿人皮的蒲團上。
他的臉色蒼白如紙，額頭上佈滿了豆大的汗珠，原本佝僂的背脊此刻更是彎得像一張弓。
背後那第三隻手，此刻正瘋狂地抓撓著地面，留下一道道深深的爪痕。

「可惡……可惡……」
趙執事咬牙切齒，眼中滿是不甘與憤怒。
「我的分身……我花了三百年培育的分身……就這麼沒了？」

他永遠忘不了幾個小時前的那一刻。
當他正在練功的時候，突然感覺到一股劇痛從心口傳來，彷彿有什麼東西被人生生撕裂。
那是他和分身之間的聯繫斷裂的感覺。
那一瞬間，他差點直接走火入魔。

「是那個小子……一定是那個小子……」
趙執事的眼中閃過一絲瘋狂的光芒。
「除了他還能有誰？他是怎麼做到的？他到底是什麼人？」

他站起身，踉蹌了幾步。
分身被殺帶來的反噬讓他的修為倒退了整整一個小境界，現在的他只有築基初期的戰鬥力。
這對於他來說是一個巨大的打擊。

但這並不意味著他就此認輸。
恰恰相反，這讓他更加確定——那個叫李默的小子，身上一定藏著什麼驚天的秘密。
如果能把這個秘密搶過來……

「等著吧，小子。」
趙執事舔了舔乾裂的嘴唇，眼中閃爍著貪婪的光芒。
「等我恢復過來，我會親手把你的秘密挖出來。」
「到時候……你會求我殺了你的。」

黑暗中，趙執事的笑聲陰森而瘋狂。
像是來自地獄的催命符。

（本章完）
