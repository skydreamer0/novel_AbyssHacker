# 內核 2.0 功能設計 (Kernel 2.0 Features)

> **解鎖時機**：第一卷末「硬重啟」後
> **主題**：從「生存」升級為「掌控」

---

## 核心升級

### 1. 介面升級
- **視覺風格**：從藍綠霓虹升級為更精緻的全息投影
- **新增HUD元素**：
  - 小地圖（可標記已探索區域）
  - 快捷技能欄
  - 威脅雷達（標記敵對目標方向）

### 2. 【數據備份】(Data Backup) 🆕
- **功能**：可保存當前技能/功法的「快照」
- **用途**：
  - 學習新功法前備份，失敗可回滾
  - 保存特定狀態供分析
- **限制**：同時只能保存3個快照，覆蓋需消耗能量

### 3. 【遠程掃描】(Remote Scan) 🆕
- **功能**：解析之眼升級，可遠程掃描
- **範圍**：從接觸距離提升至100米
- **用途**：
  - 偵查看不到的敵人
  - 分析遠處的陣法結構
- **限制**：距離越遠，精度越低，能量消耗越高

### 4. 【協議分析】(Protocol Analysis) 🆕
- **功能**：可解析更複雜的靈力結構和通訊協議
- **用途**：
  - 分析築基期以上的功法
  - 解密加密的傳訊玉簡
  - 破解更高級的陣法
- **進化方向**：為未來「駭入世界源代碼」做鋪墊

### 5. 【能量壓縮】(Energy Compression) 🆕
- **功能**：將能量壓縮存儲，提高能量上限
- **效果**：能量上限從1000提升至1500
- **代價**：壓縮過程需要專注，無法戰鬥時使用

### 6. 【應急協議】(Emergency Protocols) 🆕
- **功能**：預設多種危機應對方案
- **已解鎖**：
  - 【自動防禦】：受到攻擊時自動啟動防護
  - 【假死模式】：極限情況下模擬死亡狀態
- **待解鎖**：
  - 【數據轉移】：瀕死時將意識數據備份

## 被動升級

### 防火牆 V2.0

### 生物反應爐 V2.5
- **強化**：轉化效率提升50%
- **新增**：可「標記」高價值目標，擊殺後額外獲得能量

## 隱藏功能（待解鎖）

### 【白光協議】(White Light Protocol)
- **觸發條件**：遭遇深淵母神級威脅
- **效果**：釋放對深淵具有克制作用的能量
- **代價**：極高能量消耗 + 暴露自身特殊性
- **謎團**：這個協議的來源是什麼？

### 【源代碼存取】(Source Code Access)
- **解鎖條件**：未知（可能需要舊時代遺跡的鑰匙）
- **效果**：可以查看和修改這個世界的「底層規則」
- **限制**：每次存取都會被「世界意志」察覺
