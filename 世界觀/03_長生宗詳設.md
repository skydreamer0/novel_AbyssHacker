# 長生宗詳設

> **最後更新**：第二卷第三十章後

---

## 1. 宗門真面目

- **表面**：傳統修仙宗門，追求長生不老。
- **真相**：整個宗門是深淵母神的**育種場**。所有弟子都是「深淵種子」的培養料。
- **「飛升」**：所謂的飛升儀式，實際上是將修煉到一定程度的弟子獻祭給深淵母神。
- **高層長生的代價**：長老級以上修士與深淵母神簽訂契約，成為「祂」在世界的代言人，以此換取「長生」（實為持續異變但保持自我意識）。

## 2. 組織結構

```
掌門（元嬰期，從未露面，已「不可名狀」化）
│
├── 內門
│   ├── 五名長老（築基期以上）
│   │   ├── 大長老
│   │   ├── 二長老
│   │   ├── 三長老（周長老）← 與外門有秘密交易
│   │   ├── 四長老
│   │   └── 五長老
│   │
│   ├── 執法堂 ← 負責清理叛徒與「廢品」
│   │   └── 周通（金袍老者，副堂主）
│   │
│   ├── 煉器堂
│   │   └── 第七分部 ← 主角潛伏處
│   │
│   └── 內門弟子
│       └── 紅綾大師姐（詳見角色檔案）
│
└── 外門
    ├── 四名執事（築基期）
    │   ├── 趙執事（已死）— 築基中期，管理東區
    │   ├── 錢執事 — 築基後期，擅長追蹤，與趙執事有血緣關係
    │   ├── 孫執事 — 膀大腰圓的胖子，精明
    │   └── 李執事 — 最年輕但最強
    │
    └── 外門弟子
        ├── 普通弟子
        └── 雜役弟子（最底層）
```

## 3. 重要地點

| 地點 | 說明 |
|------|------|
| **外門弟子居住區** | 雜役弟子居住的破舊區域 |
| **深淵區** | 外門最危險的區域，專門清理深淵污染物，死亡率極高 |
| **後山** | 變異獸出沒的禁地，也是主角的狩獵場 |
| **執事殿** | 外門四執事的辦公場所 |
| **煉器堂第七分部** | 主角第二卷的主要活動地點 |
| **內庫** | 位於廢料庫下方 150 米。表面是物資存放地，實際上是「孵化場」——培育「長生丹」的深淵生物飼養場/囚禁地 |

## 4. 天樞協議

長生宗的統一通訊系統，連接宗門內所有設施。

| 項目 | 內容 |
|------|------|
| **名稱** | 天樞協議 |
| **功能** | 連接陣法、法器、清理者傀儡、身份令牌 |
| **覆蓋範圍** | 長生宗內門與外門全域 |
| **已知漏洞** | 與外門陣法同源，可被主角的駭入協議偽造指令 |
| **劇情意義** | 主角駭入天樞協議是第一卷的關鍵轉折 |

## 5. 宗門內的資訊階層

> 不同等級的修士能接觸到不同程度的「真相」。

| 等級 | 知道什麼 | 不知道什麼 |
|------|----------|-----------|
| **雜役弟子** | 只知道「好好修煉就能飛升」 | 一切真相 |
| **普通外門弟子** | 知道宗門有危險區域 | 不知道修煉的真正代價 |
| **外門執事（築基期）** | 知道部分內門秘密、清理者的存在 | 不知道宗門與深淵母神的契約 |
| **內門長老（金丹期）** | 知道獻祭的存在、與深淵的關係 | 可能不完全清楚掌門的計畫 |
| **掌門（元嬰期）** | 知道全部真相 | — |
| **主角（系統觀測者）** | 能看透幻覺，碎片化了解真相 | 尚未掌握完整的歷史脈絡 |
